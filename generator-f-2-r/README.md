## Installation

First, install [Yeoman](http://yeoman.io) and generator-f-2-r using [npm](https://www.npmjs.com/) (we assume [node.js](https://nodejs.org/) is already installed).

```bash
npm install -g yo
npm install -g generator-f-2-r
```

Then generate the new example:

```bash
yo f-2-r:example
```
